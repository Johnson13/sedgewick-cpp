# ex.4.12

## task

en: Convert to postfix the expression

    (5 * ((9 * 8) + (7 * (4 + 6))))

ru: Преобразуйте в постфиксное выражение:

    (5 * ((9 * 8) + (7 * (4 + 6))))

## solution

```
(
5  5
*     *
(     *
(     *
9  9  *
*     * *
8  8  * *
)  *  *
+     * +
(     * +
7  7  * +
*     * + *
(     * + *
4  4  * + *
+     * + * +
6  6  * + * +
)  +  * + *
)  *  * +
)  +  *
)  *

5 9 8 * 7 4 6 + * + *
```
